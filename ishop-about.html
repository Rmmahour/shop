About Ishop Model

Ishop website is a PHP/MySQL based catalog with shopping cart application that has been incorporated with extensive feature list and is one of the most flexible shopping cart solutions. It has a wide range of customization options for e-commerce website owners and clients.

This e commerce solution is so flexible that business websites can either be directly integrated into an existing website - or a new website can be built around the main ishop e-commerce engine.

The main Ishop website is built with the powerful PHP web scripting language and uses a fast MySQL database server for the online store data. The combination of PHP and MySQL allows ishop website to run on any webserver environment that supports PHP and MySQL, which includes Linux, Solaris, BSD, Mac OS X, and Microsoft Windows environments.
